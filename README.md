# ao

ao, pronouncing as A-O, is a tiny interpreter language.
Its runtime gets implemented in only 500+ lines of code.

It supports:

* JSON types: `null`, `true`, `false`, `0`, `1.0`, `"string"`, `[1, 2]`, `{"key": "value"}`.
* Assignment: `x = 1;`.
* Function: `double = f (a) { return a * a; };`.
* Arithmatic: `+`, `-`, `*`, `/`, `%`, `<<`, `>>`, `&`, `|`, `~`.
* Comparasion: `==`, `!=`, `>`, `>=`, `<`, `<=`.
